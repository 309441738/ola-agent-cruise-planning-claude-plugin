# Cruise Planning Agent · Claude Code 插件

An intelligent assistant designed to help you plan your cruise vacations seamlessly. Get insights, recommendations, and booking assistance.

![published with craftwave](https://img.shields.io/badge/published%20with-craftwave-6366f1)

## 安装

在 Claude Code 中执行：

```
/plugin marketplace add OWNER/ola-agent-cruise-planning-claude-plugin
/plugin install ola-agent-cruise-planning@ola-agent-cruise-planning-marketplace
```

> 将 `OWNER` 替换为本仓库所属的 GitHub 账号/组织名。

安装后即可在 Claude Code 里使用技能 **Cruise Planning Agent**。本仓库由
[CruiseSkillBridge](https://cruiseskillbridge.com) 按 Claude Code 插件市场规范自动生成。
